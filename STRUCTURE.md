# Complete file structure created. Move these files to proper Next.js app directory structure:

## Root level:
- package.json
- tailwind.config.ts
- tsconfig.json
- next.config.js
- postcss.config.js
- .gitignore
- .prettierrc
- README.md

## App directory (app/):
- layout.tsx
- page.tsx
- globals.css

## Components (can be in root or components/ folder):
- Hero.tsx
- PosterSection.tsx
- About.tsx
- IssueGrid.tsx
- CallForSpeakers.tsx
- FormSection.tsx
- FinalCTA.tsx
- Footer.tsx

## Public directory:
- poster.jpg (add your poster image)

## Installation:
1. Create directory structure as shown above
2. Move files to correct locations
3. Run: npm install
4. Update FormSection.tsx with your Google Form URL
5. Add poster image to public/
6. Run: npm run dev
